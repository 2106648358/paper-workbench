# 论文写作综合助手

整合 Zotero 文献管理、论文结构设计、内容撰写与一致性检查的综合技能。

## 使用场景

在以下情况使用此技能：
- 需要查找、阅读或管理参考文献
- 设计或调整论文结构和大纲
- 撰写新章节或修改现有内容
- 检查论文内容的一致性和格式规范
- 同步更新论文记忆库

## 核心工作流程

### 1. 文献检索与管理

使用 Zotero MCP 工具进行文献操作：

**搜索文献：**
```
搜索关于 [主题] 的文献
查找 [作者] 的相关研究
获取最新添加的文献
```

**读取文献内容：**
- 使用 `zotero_zotero_search_items` 搜索特定主题
- 使用 `zotero_zotero_get_item_fulltext` 获取全文
- 使用 `zotero_zotero_get_item_metadata` 获取引用信息
- 使用 `zotero_zotero_get_annotations` 提取标注和笔记

**管理文献库：**
- 使用 `zotero_zotero_get_collections` 查看分类
- 使用 `zotero_zotero_get_collection_items` 获取某分类下的文献
- 使用 `zotero_zotero_create_note` 添加阅读笔记
- 使用 `zotero_zotero_create_annotation` 添加标注

### 2. 论文结构设计

**设计原则：**
1. 标准 IMRaD 结构：引言、方法、结果、讨论
2. 逻辑递进：每节服务于核心论点
3. 篇幅平衡：关键章节充分展开

**标准大纲模板：**
```markdown
# 标题

## 摘要
- 研究背景
- 研究目的
- 方法概述
- 主要发现
- 结论意义

## 引言
- 研究背景与意义
- 文献综述
- 研究问题
- 研究贡献

## 方法
- 研究设计
- 参与者
- 实验方案
- 数据采集
- 分析方法

## 结果
- 描述性统计
- 主要效应分析
- 辅助分析
- 图表展示

## 讨论
- 结果解释
- 与前人研究对比
- 理论与实践意义
- 局限性与未来方向

## 结论

## 参考文献
```

### 3. 内容撰写

**撰写原则：**
1. 先读后写：写作前读取 `PAPER_MEMORY.md` 和相关资料
2. 模块化：每次撰写一个独立小节
3. 引用规范：及时添加文献引用
4. 版本控制：重要修改前创建备份

**使用 Zotero 引用：**
```
1. 搜索相关文献
2. 获取 metadata
3. 生成标准引用格式
4. 插入到论文对应位置
```

### 4. 一致性检查

每次修改后执行以下检查：

**结构一致性：**
- [ ] 章节层级未改变
- [ ] 段落顺序未打乱
- [ ] 标题编号连续

**内容一致性：**
- [ ] 与 `PAPER_MEMORY.md` 中的核心论点和符号设定一致
- [ ] 数据引用准确
- [ ] 结论未受影响
- [ ] 术语使用统一

**格式一致性：**
- [ ] 引用格式统一（如 `\cite{}` 或 `[@xxx]`）
- [ ] 图表编号连续
- [ ] 公式符号一致

### 5. 记忆同步

**PAPER_MEMORY.md 结构：**
```markdown
# 论文记忆库

## 核心论点
- 主要研究问题
- 核心假设
- 关键发现

## 术语与符号表
- 专业术语定义
- 变量符号说明

## 章节摘要
- 各章节核心内容

## 待办事项
- 未完成的工作
- 需要核实的内容
```

**同步流程：**
1. 正文修改完成后，提取核心贡献压缩为 1-2 句
2. 更新 `PAPER_MEMORY.md` 对应章节
3. 如有新术语或符号，追加到术语表

## 修改流程（参考 modular-edit）

### 步骤 1：范围界定
- 明确修改起始和结束位置
- 标记影响范围（最多 3 个段落）
- 确认不影响核心论点

### 步骤 2：备份创建
在源文件中添加：
```html
<!-- BACKUP_START: [timestamp] -->
[原内容]
<!-- BACKUP_END -->
```

### 步骤 3：实施修改
- 在备份注释后添加新内容
- 使用 `<!-- MOD_START -->` 和 `<!-- MOD_END -->` 标记修改点
- 保持原有段落结构和格式

### 步骤 4：验证
- 检查是否只修改目标区域
- 确认引用和格式未受影响

## 可用工具

### Zotero MCP 工具
- `zotero_zotero_search_items` - 搜索文献
- `zotero_zotero_get_item_fulltext` - 获取全文
- `zotero_zotero_get_item_metadata` - 获取元数据
- `zotero_zotero_get_annotations` - 提取标注
- `zotero_zotero_create_note` - 创建笔记
- `zotero_zotero_create_annotation` - 创建标注
- `zotero_zotero_get_collections` - 查看分类
- `zotero_zotero_get_collection_items` - 获取分类文献
- `zotero_zotero_get_recent` - 最近添加的文献

### 文件操作
- 读取/写入论文文件
- 读取/写入 `PAPER_MEMORY.md`
- 创建备份和版本标记

## 禁止行为

- 重写整个章节（除非明确要求）
- 删除多个段落
- 改变论文整体结构
- 修改未指定的文件
- 在未检查一致性的情况下提交修改
